The main result is that rent is overwhelmingly driven by `sq_ft`, with `bedrooms` and `bathrooms` adding secondary signal but also showing substantial multicollinearity. Location, year built, parking, and pet-friendliness are weak predictors in this dataset once size is controlled for. On validation, linear regression and ridge were strong and very similar (`R²` about 0.91 on the test split), while random forest was only modestly better in cross-validation and roughly tied on the held-out test set. A log-linear variant improved residual diagnostics somewhat but materially hurt predictive accuracy.

I also checked assumptions explicitly: the linear model shows non-normal residuals, heteroscedasticity, elevated VIFs, and a nontrivial number of influential points, so coefficient-level inference should be treated cautiously even though predictive performance is good.
